# Norwegian UI Standards

## Language

- All patient-facing UI text in Norwegian Bokmal (nb-NO)
- Code comments and variable names in English
- HTML lang attribute: `lang="nb"`
- Use "du/deg" (informal), never "De/Dem" (formal)

## Formatting

- Date: DD.MM.YYYY (e.g., 02.03.2026)
- Time: HH:MM 24-hour (e.g., 14:30)
- Currency: NOK with space as thousands separator, comma as decimal (1 234,50 kr)
- Phone: +47 XX XX XX XX

## Accessibility (WCAG 2.1 AA — required by Norwegian law)

- All form inputs must have visible labels in Norwegian
- Error messages must explain the problem AND the fix, in Norwegian
- Color contrast minimum 4.5:1 for normal text, 3:1 for large text
- All functionality must be keyboard accessible
- Images must have descriptive alt text in Norwegian
- Skip navigation links must be present

## Medical Terminology

- Use Norwegian terms first, Latin/English in parentheses for clinical terms
- Kiropraktor, Behandling, Undersokelse, Pasientjournal, Timebestilling
- Henvisning, Epikrise, Rontgen, MR
- ICPC-2 descriptions: use Helsedirektoratet's official Norwegian translations
