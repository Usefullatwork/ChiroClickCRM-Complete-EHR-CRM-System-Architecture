# Kliniske Presentasjons-Maler (Clinical Presentation Templates)

## 1. AKUTT KORSRYGG (Acute Low Back Pain)

### Anamnese Template
```
**Yrke:** [Arbeid med fysisk belastning/kontor]
**Presentasjon:** Akutt vondt i korsryggen, vart [tidspunkt]
**Debut:** Kom etter [løft/vridning/fall/ukjent årsak]
**Lokalisasjon:** Kjenner det [hø/ve/midten] side i korsryggen [ved PSIS/SI-ledd/paraspinalt]
**Stråling:** [Ned i nates/lår/legg / Ingen utstråling]
**Forverrende:** [Sitting/standing/fleksjon/rotasjon/om morgenen]
**Lindrende:** [Bevegelse/hvile/varme/sm.stillende]
**Tidligere:** [Første gang / Har hatt tidligere i [år] / Prolaps historikk]
**Bildediagnostikk:** [MR/RTG dato og funn / Ikke tatt bilder]
**B&B:** [Normal / Treg mage / Tisse oftere]
**MEDS:** [Paraseth/Ibux/Voltaren/Tramadol / nei]
**Tidligere behandling:** [KFA/PT/DC / Første gang]
```

### Undersøkelse Template
```
**Observasjon:**
- Antalgisk holdning [hø/ve listing]
- [Gibbus mot ve/hø ved fleksjon]

**ROM:**
- Red fleksjon/ekstensjon/lateral bøy mot [side]

**Nevrologisk:**
- Reflekser: [+2/+1/0] patella, [+2/+1/0] achilles
- SLR: [neg / pos ve/hø, [grader]]
- Styrke: [Svak ve/hø L5 / Sterk i bena]
- Sensibilitet: [neg pw og vib / red pw [område]]
- Hæl/tågange: [OK / Svakere hø/ve]

**Palpasjon:**
- Lås [hø/ve] IS ledd
- Lås [segment] [retning]
- Hyperton [hø/ve] ES/QL
- Stram [hø/ve] psoas/hamstring/hofte

**Spesielt:**
- [Kort hø/ve ben]
- [Ve/hø konveks lumbal kurve]
```

### Behandling Template
```
**Bløtvev:** Massasje [ES/QL/hofte mm/begge sider]
**Mobilisering:**
- P-R [ISD/fs/ss/kick/drop]
- P-L [fs/ss/ISD]
- [Segment] [retning]
**Ekstremitet:** [hø/ve ben fs/ss]
**Øvelser:** [Strikk/McKenzie/Stabilitetsøvelser]
**Råd:** [Ergonomi/varme/aktivitet]
**Tape:** [Kinesiotape korsrygg hvis indisert]
```

### Plan Template
```
**Diagnose:** Akutt lumbago [med/uten utstråling]
**Prognose:** [God/moderat/dårlig], forventer bedring [tidslinje]
**Oppfølging:** [1-2 ggr/uke i 2-3 uker / etter behov]
**Hjemmeoppgaver:** [Spesifikke øvelser gitt]
**Røde flagg:** [Ingen / Henvis hvis forverring av nevrologiske symptomer]
```
